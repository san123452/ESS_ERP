# 🔋 ESS 배터리 제조 공정 통합 관리 시스템

> 디지털 BOM 기반의 배터리 제조 공정을 통합 관리하고
> AI + Python + RPA를 활용하여 데이터 처리 및 자동화를 구현한 프로젝트

---

## 📖 프로젝트 소개

ESS 배터리 제조 공정에서 발생하는 데이터를 통합 관리하고,
기존 수작업 중심의 데이터 처리 방식의 한계를 개선하기 위해
**ERP 형태의 통합 시스템을 구축한 프로젝트입니다.**

특히 **BOM 구조 기반 데이터 관리 + RPA 자동화 + Python 데이터 처리**를 통해
업무 효율성과 데이터 정확성을 향상시키는 것을 목표로 했습니다.

## ⚙️ 주요 기능

* 배터리 제조 공정 데이터 통합 관리
* BOM 구조 기반 데이터 처리

---

## 🔄 시스템 흐름 (Process)

전자공시 / 제조 데이터 수집
↓
RPA 자동화 수집
↓
Python 데이터 정제 (Pandas)
↓
BOM 기반 데이터 구조화
↓
ERP 시스템 관리
↓
Excel 분석 및 시각화
